
This project consists of 3 parts, to run it please check the prerequisites:

	1.To call solr API for full text search -> need to setup solr locally because there is security check so you cannot connect to remote solr
	2.To call Rightnow API for FAQ search -> need to use 'clear-guest' network
	3.To call DBCS for suggest keywords -> need to use 'clear-guest' network
